# H5-CSS3-J5

## 技术说明
1. 慕课网作业；
2. 采用HTML5作为项目的结构层；
3. 采用CSS3完成项目的样式；
4. 采用CSS3实现项目的动画特效；
5. 采用原生JavaScrip控制交互效果。

## 功能说明
1. 打开网页第一页动画自动播放，顶部导航条逐渐显示；
2. 滚动鼠标，随着页面位置的变化对应出现在屏幕中的页开始播放动画；
3. 鼠标滚动到一定高度时，右侧导航条渐显、顶部导航条改变样式；
4. 屏幕当前页对应的顶部导航项和右侧导航项改变样式；
5. 顶部导实现航滑动门效果；
